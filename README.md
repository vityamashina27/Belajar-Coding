# Installasi
Clone git repo ke device kalian setelah clone ketik code dibawah dalam git/cmd kalian
```
  npm install
  npm run start
```

Run dengan membuka browser kalian lalu ketik url 
``` 
  localhost:3000
  atau
  127.0.0.1:3000
```
